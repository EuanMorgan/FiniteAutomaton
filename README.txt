My answer to this coursework is split across several classes, each named accordignly
The program should be run using the App class
You can pass the following arguments to run the code for each question

-c (Complement - followed by 1 .txt filename)

-i (Intersection - followed by 2 .txt filenames)

-s (Symmetric Difference - followed by 2 .txt filenames)

-n (Non-emptyness: followed by 1 .txt filename)

-e (Equivaence: followed by 2 .txt filenames)

Examples:

java App -c D1.txt

java App-e D1.txt D2.txt
